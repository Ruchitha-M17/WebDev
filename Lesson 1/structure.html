HTML Structure/Boilerplate

The HTML Boilerplate includes the essential HTML code structure and also includes some of the commonly used meta tags, links to CSS and JavaScript files, and some of the other elements required for creating a web page.

DOCTYPE:
The first line in your HTML code should be the doctype declaration. A doctype tells the browser what version of HTML the page is written in.
The <html> tag:
Represents root/top level element of the HTML document. All other elements are the descendents of this element.
The lang="en" attribute specifies the language of the document is English.
The <head> tag:
Contains machine readable information or the metadata. Is palced between the <html> tag and the <body> tag.
The <meta charset="UTF-8"> is an attribute for the UTF-8 standard character encoding you should use in your web pages.
The <meta name="viewport" content="width=device-width, initial-scale=1.0"> tag renders the width of the page to the width of the device's screen size. The initial-scale controls the zoom level. The value of 1 for the initial-scale prevents the default zoom by browsers.
It also contains other tags that links CSS and JS files, favicons, style tags, fonts to properly render the web page.
The <title> tag:
Defines the document's title that is dispalyed on the webpage's tab.
<body> tag:
Represents the content of the entire HTML document.
